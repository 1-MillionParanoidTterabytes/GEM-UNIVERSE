Mirror is a MMO Scale Networking library for Unity, used in uMMORPG, uSurvival
and several MMO projects in development.

  https://github.com/vis2k/Mirror

This Asset:
  * Configures the Mirror DLLs for use in Unity
  * Contains all the example Projects

Documentation:
  Mirror is based on UNET.
  Please use the UNET Manual: https://docs.unity3d.com/Manual/UNet.html
  And use 'using Mirror' instead of 'using UnityEngine.Networking'
